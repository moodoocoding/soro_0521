# SORO 학생 연간 이벤트 공모전 - 작업 노트 (notes.md)

## 2026. 05. 26. 주요 작업 및 업데이트 내역

### 1. UI 및 텍스트 수정
- **픽셀아트 진입 버튼 변경**: "직접 도트 찍기로 전환" 텍스트를 "픽셀아트 에디터"로 보다 직관적으로 변경.
- **서비스 명칭 범용화**: 시스템 알림 및 팝업창에서 노출되던 "구글 스프레드시트" 단어를 모두 "클라우드"로 통일하여 교체.
- **공모 기간 업데이트**: 키링 공모전 안내 요강의 공모 기간을 "2026. 6. 1.(월) ~ 2026. 6. 22.(월)"로 명확히 수정.

### 2. 레이아웃 및 반응형 스타일(CSS) 오류 해결
- **태블릿 픽셀아트 화면 잘림 현상(버그) 수정**: 
  - 태블릿 환경(가로 모드 등)에서 픽셀아트 에디터가 화면 하단만 보이고 위로 밀려 렌더링되던 레이아웃 충돌 해결.
  - 원인: 모달 애니메이션(`transform`)과 절대 위치(`position`) 간의 충돌 및 100vh 인식 오류.
  - 조치: 픽셀 에디터 전체 화면 시 `transform: none !important`로 애니메이션을 해제하고, `height: 100dvh` 속성을 적용해 모바일/태블릿에서 화면에 꽉 차게 고정되도록 CSS 수정.

### 3. 공모전 데이터(app.js) 최적화 및 동적 기능 확장
- **동적 '예시 작품' 탭 구현**: 컷툰 공모전 갤러리를 `index.html`에 하드코딩하지 않고, `app.js`의 `CONTESTS_DATA` 내에 예시 이미지(`cuttoon1~4.png`) 배열을 추가. 데이터가 존재할 때만 동적으로 "예시 작품 👀" 탭과 갤러리 UI가 자동 생성되도록 고도화.
- **불필요한 심사 기준표 삭제 및 탭 숨김**: '온라인 도서관', '디지털 필사', '우정사진' 공모전의 세부 심사 기준표 데이터(`evaluationCriteria`)를 삭제. 더 나아가 심사 기준 데이터가 없는 공모전 모달을 열 때는 '심사 기준표' 탭 버튼이 아예 노출되지 않도록 UI 렌더링 로직 개선.
- **특별 혜택 안내 추가**: 위 3개 공모전(도서관, 필사, 사진)의 공모 요강(rules)에 *"특별 혜택: 학급 인원의 2/3 이상이 참여하였을 경우 학급 전체에 소정의 기념품을 지급합니다."* 라는 문구 일괄 추가.
