# SORO 학생 연간 이벤트 공모전 - 작업 노트 (notes.md)

## 2026. 05. 26. 주요 작업 및 업데이트 내역

- **서비스 명칭 범용화**: 시스템 알림 및 팝업창에서 노출되던 "구글 스프레드시트" 단어를 모두 "클라우드"로 통일하여 교체.
- **공모 기간 업데이트**: 키링 공모전 안내 요강의 공모 기간을 "2026. 6. 1.(월) ~ 2026. 6. 22.(월)"로 명확히 수정.
- **키링 공모전 Canva 템플릿 링크 추가**: 키링 공모전 안내 요강 하단에 Canva 템플릿 참고 링크(`https://canva.link/7al5bl4pt23j015`)를 삽입하고 클릭 시 새 창에서 열리도록 앵커 링크 스타일(`.contest-link`)을 `style.css`에 추가.

### 2. 레이아웃 및 반응형 스타일(CSS) 오류 해결
- **태블릿 픽셀아트 화면 잘림 현상(버그) 최종 수정**: 
  - 태블릿 환경(가로 모드 등)에서 픽셀아트 에디터가 화면 위로 밀려 윗부분이 잘리고 아래쪽에 거대한 빈 공간이 생기던 레이아웃 버그 최종 해결.
  - 원인: 서랍(Drawer) 양식이 길어 스크롤을 내린 상태(`scrollTop > 0`)에서 에디터가 활성화될 경우, 부모 컨테이너인 `.drawer-content`의 스크롤 위치가 유지되어 에디터 전체가 상단으로 밀려 렌더링됨. 또한 `transform`이 적용된 조상 돔 구조에 의해 fixed 포지셔닝이 깨짐.
  - 조치: 
    1. 에디터가 활성화(`isDraw === true`)될 때 자바스크립트로 부모 서랍 컨테이너(`.drawer-content`)의 스크롤을 맨 위로 초기화(`scrollTop = 0`) 처리.
    2. `.pixel-editor-shell`을 `position: absolute`로 전환하여 부모 transform 레이아웃 안에서 정적/안정적으로 매핑되도록 조정.
    3. `.drawer.pixel-fullscreen .drawer-content`에 `height: 100% !important`, `overflow: hidden !important`를 명시하여 스크롤 발생을 완전히 억제하고 뷰포트에 꽉 차게 정렬.

### 3. 공모전 데이터(app.js) 최적화 및 동적 기능 확장
- **동적 '예시 작품' 탭 구현**: 컷툰 공모전 갤러리를 `index.html`에 하드코딩하지 않고, `app.js`의 `CONTESTS_DATA` 내에 예시 이미지(`cuttoon1~4.png`) 배열을 추가. 데이터가 존재할 때만 동적으로 "예시 작품 👀" 탭과 갤러리 UI가 자동 생성되도록 고도화.
- **불필요한 심사 기준표 삭제 및 탭 숨김**: '온라인 도서관', '디지털 필사', '우정사진' 공모전의 세부 심사 기준표 데이터(`evaluationCriteria`)를 삭제. 더 나아가 심사 기준 데이터가 없는 공모전 모달을 열 때는 '심사 기준표' 탭 버튼이 아예 노출되지 않도록 UI 렌더링 로직 개선.
- **특별 혜택 안내 추가**: 위 3개 공모전(도서관, 필사, 사진)의 공모 요강(rules)에 *"특별 혜택: 학급 인원의 2/3 이상이 참여하였을 경우 학급 전체에 소정의 기념품을 지급합니다."* 라는 문구 일괄 추가.

### 4. 내 접수 내역 서랍 확장 및 레벨/스테이지 시스템 구축
- **서랍 너비 확대**: 내 접수 내역 확인 서랍(`lookup-drawer`)의 너비를 기존 540px에서 800px(모바일에서는 100%)로 대폭 확장하여, 정보 표현력을 크게 향상시키고 쾌적한 화면 정렬 완성.
- **게이미피케이션 기반 레벨링 시스템 구현**: 고유 제출한 공모전 개수를 카운팅하여 7개의 레벨 단계(Lv.0 ~ Lv.6)로 나누어 학생들에게 직관적으로 레벨명과 설명이 매핑되도록 설계.
- **실시간 프로그레스 바(XP Gauge) 적용**: 미션 완료율(0% ~ 100%)에 따라 작동하는 세련된 블루/퍼플 그라데이션 게이지 바를 추가하고, 서랍이 열릴 때 게이지바가 동적으로 충전되는 미크로 인터랙션 모션 제공.
- **미션 달성 보드(6열 그리드) 탑재**: 6종의 연간 공모전을 한눈에 보는 보드판으로 렌더링. 완료된 미션은 활성 링(🟢) 및 파란 테두리로 표시되고, 미제출 미션은 잠금 상태(🔒)로 회색 음영 처리되어 지속적인 참여 동기를 부여.
- **로그아웃 시 대시보드 리셋**: 로그아웃 시 레벨 카드가 흔적 없이 정리되도록 세션 초기화 로직 보강.

### 5. 공모전 활성화 관리 (키링 공모전 단독 오픈)
- **키링 공모전만 단독 활성화**: 메인 페이지 및 시스템 전체에서 현재 6월 기준으로 "키링 공모전"만 "접수 중" 상태가 되도록 다른 공모전의 활성월(`activeMonths` 등)을 격리하고 강제 활성화(`FORCE_ACTIVE_CONTESTS`) 배열을 청소 완료.
